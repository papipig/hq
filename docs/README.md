# Game rules and player interface

Player will play turn by turn. They can:
* perform action THEN move
OR
* move THEN perform action

Actions can be:
* Action 1: Attack
* Action 2: Cast a spell
* Action 3: Search for treasure
* Action 4: Search for secret doors
* Action 5: Search for traps
* Action 6: Disarm a trap
